# MacVaporApp
MacOS sample app that is also a Vapor REST server. This is the basecode for following post [Integrating Vapor into Native macOS Apps](https://javios.eu/vapor/integrating-vapor-into-native-macos-apps/)

## Requirements

- **Xcode 26.2 or later**
- **macOS 26.5.1 or later**
